# Changelog

## [unreleased]

### Новое
- Стиль «Fallout» — ретро-терминал PIP-Boy: CRT-эффекты (scanlines, виньетка, мерцание), монохромная обложка, верхняя панель с вкладками, прогресс-бар, спектроанализатор
- Fallout: водяной знак pip-boy.png в инфо-панели, BITRATE в нижней строке, батарея привязана к высоте

### Исправлено
- Приоритет источников: играющие выше пауз, со звуком выше без звука
- Twitch теперь корректно определяется как играющий
- Spotify Canvas-клипы больше не сбивают тайминг и не скрывают обложку
- Обложка автоматически перезагружается, если не загрузилась с первого раза
- Исправлена ошибка `Extension context invalidated` при закрытии вкладки
- Улучшено определение состояния воспроизведения для сайтов без MediaSession API

## [2.1.1] - 2026-07-20

### Исправлено
- Стиль «Минималистичный»: исправлена прокрутка длинных названий, обрезание хвостиков букв, дублирование текста в режиме ожидания

## [2.1.0] - 2026-07-19

### Новое
- Модульная система стилей — 5 оформлений на выбор: классический, компактный, стеклянный, минималистичный, магнитола
- Стиль «Магнитола» (stereo) — стилизованный светодиодный дисплей с прокруткой текста
- Стиль минимал — без фона (только название и исполнитель)
- Управление стилями из Pop-up меню расширения в браузере
- Предпросмотр виджета в Pop-up меню расширения в браузере
- Возможность использовать разные стили для разных сцен через Custom CSS в OBS (`:root { --widget-style: '...' }`)

### Исправлено
- Улучшена обработка длинных названий — анимация работает плавнее
- Исправлено зависание виджета при перезагрузке OBS (heartbeat 5с)
- Исправлена ситуация, когда данные с закрытой вкладки повторно добавляли трек

## [2.0.0] - 2026-06-14

### Главное
- Расширение работает напрямую через встроенный WebSocket-сервер OBS. Node.js и дополнительный сервер не нужны.

### Новое
- Улучшена работа с Яндекс.Музыкой
- Улучшена поддержка YouTube
- Добавлена поддержка SoundCloud
- Добавлена поддержка Spotify
- Автоматическое обнаружение музыки на всех вкладках
- Автоматическая очистка виджета при пропаже источника
- Добавлен английский язык в попап-меню, в т.ч. для инструкции настройки WebSocket сервера в OBS

### Исправлено
- Исправлена работа таймера
- Автоматическое подключение к OBS

## [1.1.0] - 2026-04-17

### Changed
- Исправлен таймер паузы, переписана анимация

## [1.0.0] - 2026-04-14

### Added
- Первый релиз расширения + сервера + виджета
- HTTP-сервер (`server.js`) для передачи данных в OBS
- Поддержка Яндекс.Музыки
- Автоматическое определение тайминга из audio элементов
- Compact/ориентация виджета
- Marquee-анимация для длинных текстов
- Color extraction из обложки
